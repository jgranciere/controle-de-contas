# Controle de Contas 💸

Aplicativo simples para controle de contas pessoais, desenvolvido com **React Native** e **Expo**.

---

## 📱 Funcionalidades

- Adição de contas por dia do mês
- Marcação de contas pagas
- Exclusão de contas
- Cálculo do valor total
- Interface simples e funcional

---

## 🛠️ Tecnologias

- React Native
- Expo
- JavaScript

---
